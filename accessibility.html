ACCESSIBILITY STATEMENT

Accessibility statement for MMK Accountants website.

Our Commitment to Accessibility

MMK Accountants Ltd is committed to ensuring that our website is accessible to all users, including those with disabilities. We strive to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA, which is the internationally recognised standard for web accessibility.

We believe that every visitor to our website should be able to access the information and services we provide, regardless of ability or the technology they use to browse the web.

Accessibility Features

We have implemented the following accessibility features across our website:

Navigation and Structure

  • A clear and consistent navigation structure is used across all pages
  • A "Skip to main content" link is provided at the top of every page, allowing keyboard users to bypass navigation
  • Breadcrumb navigation is provided on all inner pages to help users understand their location within the site
  • All pages have descriptive and unique page titles
  • Heading elements (h1 through h6) are used in a logical and hierarchical order

Keyboard Navigation

  • All interactive elements, including links, buttons, and form controls, are accessible via keyboard
  • A visible focus indicator is provided for all focusable elements
  • No keyboard traps exist on the website — users can navigate freely using the Tab and Shift+Tab keys
  • Dropdown menus are accessible via keyboard navigation

Screen Reader Compatibility

  • All images include descriptive alternative text (alt attributes)
  • Decorative images and icons are marked with aria-hidden="true" to avoid confusion
  • Form fields are properly labelled using <label> elements or ARIA attributes
  • ARIA landmarks are used to identify page regions (header, navigation, main content, footer)
  • Interactive elements have descriptive ARIA labels where standard text is not sufficient

Visual Design

  • Sufficient colour contrast ratios are maintained between text and background colours
  • Text can be resized up to 200% without loss of content or functionality
  • Content is responsive and accessible on all screen sizes, from mobile devices to large desktop monitors
  • No content relies solely on colour to convey information

Forms and Interactive Content

  • All form fields have associated labels
  • Error messages are clearly presented and associated with the relevant form field
  • Required fields are clearly identified
  • Form validation errors are communicated in an accessible manner

Known Limitations

While we strive to ensure full accessibility, there may be some areas of the website that are not yet fully accessible. We are continuously working to improve the accessibility of our website and address any outstanding issues. Known limitations include:

  • Some older PDF documents may not be fully accessible to screen readers. We are working to replace these with accessible versions
  • Some third-party content or embedded tools may not fully conform to WCAG 2.1 Level AA

Reporting Accessibility Issues

If you experience any difficulty accessing our website, or if you have suggestions for improving accessibility, we would like to hear from you. Please contact us using any of the following methods:

  • Email: mail@mmkaccountants.com
  • Phone (London): 020 3981 5373
  • Phone (Luton): 01582 726 600
  • Post: MMK Accountants Ltd, One Canada Square, Canary Wharf, London E14 5AA

When reporting an accessibility issue, please include:

  • The web address (URL) of the page where you experienced the problem
  • A description of the issue you encountered
  • The type of assistive technology you were using, if applicable (e.g., screen reader name and version)
  • Your browser and operating system

We aim to respond to accessibility feedback within 5 working days and to resolve reported issues as quickly as possible.

Enforcement Procedure

If you are not satisfied with our response to your accessibility concern, you can contact the Equality Advisory Support Service (EASS) at www.equalityadvisoryservice.com.

Technical Information

This website is built using standard HTML5, CSS3 and JavaScript. It is designed to be compatible with the following assistive technologies:

  • Screen readers (including JAWS, NVDA, and VoiceOver)
  • Screen magnification software
  • Speech recognition software
  • Keyboard-only navigation
